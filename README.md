## Installation

_To enjoy the Fun Igniter, just clone the repository and open index.html_

2. Clone the repo
   ```sh
   git clone https://github.com/Anisha7860/Fun_igniter.git
   ```
3. Navigate to the `Fun_igniter` folder
   ```sh
   cd Fun_igniter
   ```
4. Open the `index.html`
   ```sh
   open index.html
   ```
